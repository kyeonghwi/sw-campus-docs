## 🎯 Code Rules의 목적

1. **코드 품질 유지** - 일관된 코드 스타일로 가독성 향상
2. **아키텍처 일관성** - Multi Module + Layer Architecture 원칙 준수
3. **온보딩 효율화** - 신규 팀원이 빠르게 프로젝트에 적응
